# 📊 Modern Bullet Chart using Python

A modern Bullet Chart built with Python and Matplotlib featuring performance zones, target line, value marker, percentage labels, and a clean dashboard-style visualization.

## ✨ Features

- 📊 Horizontal Bullet Chart
- 🟢🟡🔴 Performance Zones
- 🎯 Target Line
- 🔵 Current Value Marker
- 📈 Percentage Label
- 🏆 Performance Status
- 📐 Grid Lines
- 🎨 Clean Dashboard Style

## 🖼️ Preview

This project creates a professional Bullet Chart similar to those used in business dashboards and analytics reports.

## 🛠️ Technologies Used

- Python 3
- Matplotlib

## 🚀 Installation

Install Matplotlib:

```bash
pip install matplotlib
```

Run the project:

```bash
python bullet_chart.py
```
